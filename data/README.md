# Data Directory

This directory contains all data files related to Croatian toponymy and family history research.

## Structure

- **shapefiles/** - GIS shapefiles of Croatian administrative boundaries, settlements, and geographic features
- **raw/** - Original, unprocessed data files (CSV, Excel, databases, etc.)
- **processed/** - Cleaned and processed data ready for analysis

## Data Sources

When adding data to this repository, please document:
- Source of the data
- Date acquired
- Any licensing information
- Processing steps applied

## Recommended Data Types

- Administrative boundaries (županije, općine, settlements)
- Historical placename records
- Census data
- Church records with location information
- Cadastral data
- Surname distribution maps
