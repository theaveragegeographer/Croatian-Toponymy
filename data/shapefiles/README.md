# Shapefiles Directory

Store GIS shapefiles and other spatial data formats here.

## Supported Formats

- Shapefiles (.shp, .shx, .dbf, .prj)
- GeoJSON (.geojson)
- KML/KMZ (.kml, .kmz)
- GeoPackage (.gpkg)
- CSV with coordinates

## Suggested Data Layers

1. **Administrative Boundaries**
   - Croatian županije (counties)
   - Općine (municipalities)
   - Naselja (settlements)

2. **Historical Data**
   - Historical administrative boundaries
   - Historic settlement locations
   - Migration routes

3. **Reference Data**
   - Rivers and water bodies
   - Topographic features
   - Transportation networks

## Data Sources

- [Croatian State Geodetic Administration](https://dgu.gov.hr/)
- OpenStreetMap
- Natural Earth Data
- Historical archives and digitization projects
