# Scripts Directory

This directory contains scripts for processing, analyzing, and visualizing Croatian toponymy data.

## Structure

- **analysis/** - Scripts for analyzing placename patterns, surname distributions, and toponymic relationships
- **processing/** - Scripts for data cleaning, geocoding, and format conversion

## Common Tasks

### Data Processing
- Cleaning and standardizing placenames
- Geocoding addresses and placenames
- Converting between spatial data formats
- Merging datasets

### Analysis
- Finding placenames related to surnames
- Analyzing spatial patterns in family names
- Historical placename changes
- Linguistic analysis of toponyms

## Development
