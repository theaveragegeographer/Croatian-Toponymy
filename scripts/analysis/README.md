# Analysis Scripts

Scripts for analyzing Croatian toponymy and family history data.

## Example Analysis Tasks

1. **Surname-Placename Matching**
   - Find settlements with names matching or similar to family surnames
   - Identify potential ancestral locations

2. **Spatial Distribution Analysis**
   - Map surname distributions across regions
   - Identify clustering patterns

3. **Historical Analysis**
   - Track placename changes over time
   - Correlate with historical events and migrations

4. **Linguistic Analysis**
   - Analyze etymological patterns
   - Identify regional dialect influences in placenames
