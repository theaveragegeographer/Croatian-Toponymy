# Processing Scripts

Scripts for data preparation and processing.

## Common Processing Tasks

1. **Data Cleaning**
   - Standardizing Croatian character encoding (č, ć, đ, š, ž)
   - Removing duplicates
   - Handling missing values

2. **Geocoding**
   - Converting placenames to coordinates
   - Reverse geocoding coordinates to locations

3. **Format Conversion**
   - Converting between spatial formats (Shapefile, GeoJSON, KML)
   - Transforming coordinate systems
   - Exporting to various formats for different GIS software

4. **Data Integration**
   - Merging multiple data sources
   - Joining tabular data with spatial data
   - Standardizing schema across datasets

## Tools and Libraries

Recommended tools for working with Croatian toponymy data:
- Python: geopandas, shapely, fiona, pyproj
- R: sf, sp, rgdal
- QGIS for interactive mapping and processing
